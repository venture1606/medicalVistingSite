# PharmaPrecise — Pharmaceutical Company Website

A modern, fully responsive visiting website built for a professional 
pharmaceutical brand. Designed to showcase the company's product catalog, 
clinical credibility, and enable direct customer engagement via WhatsApp.

## 🚀 Tech Stack

- **Framework** — Next.js 14 (App Router)
- **Language** — TypeScript (strict mode)
- **Styling** — Tailwind CSS
- **UI Components** — shadcn/ui
- **Icons** — Lucide React
- **Data** — Static JSON (no backend required)
- **Deployment** — Vercel

## 📄 Pages

- **Home** — Hero, Why Choose Us, About, Featured Products, WhatsApp CTA
- **Products** — Searchable, filterable clinical portfolio with pagination
- **Product Detail** — Image gallery, composition table, dosage protocol, tabs

## ✨ Features

- 100% static — product data driven by a single JSON file
- Mobile-first responsive design
- Floating WhatsApp quick-chat button
- Product search and category filtering
- Clean architecture with 200-line file limit enforced
- SEO-ready with dynamic metadata per product page

## 🗂️ Project Structure

Built following strict Clean Architecture principles — 
thin pages, reusable components, all data in lib/data/.

## 📦 Getting Started

git clone https://github.com/venture1606/medicalVistingSite.git
